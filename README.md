# Raw图编辑工具

## 初版完成
- 时间： 2025年7月1日
- 完成基础框架搭建
- 实现实现图片拖拽显示
- 滚轮放大缩小，可放大到像素级，通过QPainter实现像素绘制
- 完成roi填充功能，能够向raw图中选择roi区域填充自定义值
